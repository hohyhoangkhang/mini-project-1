Nhập môn web căn bản

Internet là gì?

Internet là một hệ thống thông tin toàn cầu bao gồm các mạng máy tính được liên kết với nhau.

Web là gì?

Web là một hệ thống các trang thông tin được liên kết với nhau trên Internet.

Website là gì?

Website là một tập hợp các trang web liên kết với nhau.

Trang web là gì?

Trang web là một tài liệu được tạo bằng ngôn ngữ HTML và có thể được hiển thị trong trình duyệt web.

Browser là gì?

Browser là phần mềm được sử dụng để truy cập và hiển thị các trang web trên Internet.

Server là gì?

Server là một máy tính được sử dụng để lưu trữ và cung cấp các tài nguyên hoặc dịch vụ cho các máy tính khác trên mạng.
